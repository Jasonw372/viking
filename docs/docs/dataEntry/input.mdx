# Input 输入框

`Input` 组件用于接收用户输入，支持多种样式和功能扩展。

## 使用

```ts
import { Input } from 'viking-design';
```

## 代码演示

### 基础用法

```tsx
import { Input } from 'viking-design';

<Input placeholder="请输入内容" />;
```

<Input placeholder="请输入内容" />

### 大小

```tsx
import { Input } from 'viking-design';

<Flex vertical gap={'sm'}>
  <Input size="lg" placeholder="大号输入框" />
  <Input size="sm" placeholder="小号输入框" />
<Flex/>;
```

<Flex vertical gap={'sm'}>
  <Input size="lg" placeholder="大号输入框" />
  <Input size="sm" placeholder="小号输入框" />
</Flex>

### 禁用状态

```tsx
import { Input } from 'viking-design';

<Input disabled placeholder="不可用的输入框" />;
```

<Input disabled placeholder="不可用的输入框" />

### 带图标

```tsx
import { Input } from 'viking-design';

<Input prefixIcon="user" suffixIcon="search" placeholder="带前后缀图标的输入框" />;
```

<Input prefixIcon="user" suffixIcon="search" placeholder="带前后缀图标的输入框" />

### 带前后缀

```tsx
import { Input } from 'viking-design';

<Input prepend="https://" append=".com" placeholder="带前后缀的输入框" />;
```

<Input prepend="https://" append=".com" placeholder="带前后缀的输入框" />

### 带前后缀和图标的输入框

```tsx
import { Input } from 'viking-design';

<Input prepend="https://" prefixIcon={"search"} append=".com" suffixIcon={"keyboard"} placeholder="带前后缀的输入框" />
```

<Input prepend="https://" prefixIcon={"search"} append=".com" suffixIcon={"keyboard"} placeholder="带前后缀的输入框" />



## API

### Input Props

| 属性         | 说明               | 类型                                               | 默认值  |
| ------------ | ------------------ | -------------------------------------------------- | ------- |
| disabled     | 是否禁用输入框     | `boolean`                                          | `false` |
| size         | 输入框的大小       | `'lg' \| 'sm'`                                     | -       |
| prefixIcon   | 输入框前缀图标     | `IconProp`                                         | -       |
| suffixIcon   | 输入框后缀图标     | `IconProp`                                         | -       |
| prepend      | 输入框前置内容     | `React.ReactNode`                                  | -       |
| append       | 输入框后置内容     | `React.ReactNode`                                  | -       |
| onChange     | 输入框内容变化回调 | `(e: React.ChangeEvent<HTMLInputElement>) => void` | -       |
| value        | 输入框的值         | `string`                                           | -       |
| defaultValue | 输入框的默认值     | `string`                                           | -       |

`Input` 组件还支持 `input` 元素的所有原生属性。
