# Grid 栅格

`Grid` 组件用于创建响应式的栅格布局，支持多种排列、对齐和间距设置。

## 使用

```ts
import { Grid } from 'viking-design';
const { Row, Col } = Grid;
```

## 代码演示

### 基础栅格

```tsx
import { Grid } from 'viking-design';

<Row>

      <span>col-24</span>
    </div>
  </Col>
</Row>
<Row>
  <Col span={12}>
    <div style={{ backgroundColor: '#0092ff', padding: '8px 0', textAlign: 'center', color: '#fff' }}>
```

## API

### Row Props

| 属性    | 说明         | 类型                                                                | 默认值    |
| ------- | ------------ | ------------------------------------------------------------------- | --------- |
| gutter  | 栅格间隔     | `number \| [number, number]`                                        | `0`       |
| justify | 水平排列方式 | `'start' \| 'end' \| 'center' \| 'space-around' \| 'space-between'` | `'start'` |
| align   | 垂直排列方式 | `'top' \| 'middle' \| 'bottom'`                                     | `'top'`   |

### Col Props

| 属性   | 说明               | 类型              | 默认值 |
| ------ | ------------------ | ----------------- | ------ |
| span   | 栅格占据的列数     | `number`          | -      |
| offset | 栅格左侧的间隔格数 | `number`          | -      |
| pull   | 栅格向左移动格数   | `number`          | -      |
| push   | 栅格向右移动格数   | `number`          | -      |
| xs     | \<576px 响应式栅格 | `ResponsiveValue` | -      |
| sm     | ≥576px 响应式栅格  | `ResponsiveValue` | -      |
| md     | ≥768px 响应式栅格  | `ResponsiveValue` | -      |
| lg     | ≥992px 响应式栅格  | `ResponsiveValue` | -      |
| xl     | ≥1200px 响应式栅格 | `ResponsiveValue` | -      |

`Row` 和 `Col` 组件还支持 `div` 元素的所有原生属性。
