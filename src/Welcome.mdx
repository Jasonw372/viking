# Viking Design

### 安装试试

```bash
npm install viking-design --save
```

### 使用

#### 引入样式

```tsx
import 'viking-design/dist/index.css';
```

#### 使用组件

```tsx
import { Button } from 'viking-design/';

function App() {
  return <Button btnType="primary">Hello Viking Design!</Button>;
}
```
